# MNIST Digit Classification using Neural Network

## Project Summary
This project implements a Neural Network–based image classification model to recognize handwritten digits using the MNIST dataset. The solution demonstrates an end-to-end machine learning workflow, including data preprocessing, model building, training, evaluation, and visualization using Python and TensorFlow (Keras).

## Problem Statement
Classify grayscale images of handwritten digits (0–9) into their respective categories using a supervised learning approach.

---

## Dataset Information
- Dataset: MNIST Handwritten Digits  
- Training Samples: 60,000  
- Testing Samples: 10,000  
- Image Dimensions: 28 × 28 pixels  
- Source: Loaded using Keras built-in datasets  

---

## Tools & Technologies
- Programming Language: Python  
- Frameworks/Libraries:
  - TensorFlow / Keras
  - NumPy
  - Matplotlib
- Development Environment: Jupyter Notebook  

---

## Model Architecture
- Input Layer: Flatten (28 × 28 → 784)
- Hidden Layers: Fully Connected Dense layers
- Activation Function: ReLU
- Output Layer: Softmax (10 classes)
- Optimizer: Adam
- Loss Function: Sparse Categorical Crossentropy

---

## Model Training & Evaluation
- Trained on 60,000 labeled images
- Evaluated on 10,000 unseen test images
- Performance Metrics:
  - Classification Accuracy
  - Confusion Matrix
- Achieved high test accuracy, demonstrating effective generalization
